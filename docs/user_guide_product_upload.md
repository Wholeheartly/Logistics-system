# 商品信息上传与单位转换 - 用户操作手册

## 一、功能概述

商品信息上传与单位转换模块提供以下核心功能：

1. **Excel 批量导入** - 通过上传 Excel 文件快速批量导入商品信息
2. **手动录入** - 通过表单逐条录入商品信息
3. **单位自动转换** - 系统自动将厘米(cm)转换为英寸(inch)，千克(kg)转换为磅(lb)
4. **双版本查看** - 支持在原始单位和转换单位之间快速切换查看

---

## 二、操作界面导航

进入系统后，点击左侧导航栏的 **"产品查询"** 模块，即可看到两个子页面：

- **产品搜索** - 查看已上传的商品信息
- **产品上传** - 上传新的商品信息

---

## 三、Excel 批量导入操作步骤

### 3.1 下载导入模板

1. 进入 **"产品上传"** 页面，选择 **"Excel 批量导入"** 标签
2. 点击页面右上角的 **"下载导入模板"** 按钮
3. 下载的模板包含两个工作表：
   - **产品导入模板** - 包含标准表头和示例数据
   - **填写说明** - 各字段的详细说明和填写要求

### 3.2 填写模板数据

按照模板表头填写商品信息，必填字段包括：

| 字段名 | 说明 | 是否必填 | 示例 |
|--------|------|----------|------|
| sku | 产品唯一编码 | **必填** | SKU001 |
| name | 产品名称 | **必填** | 示例产品 |
| length_cm | 长度（厘米） | **必填** | 10.0 |
| width_cm | 宽度（厘米） | **必填** | 20.0 |
| height_cm | 高度（厘米） | **必填** | 30.0 |
| gross_weight_kg | 毛重（千克） | **必填** | 1.5 |
| model | 产品型号 | 选填 | Model-A |
| specification | 规格描述 | 选填 | 10x20x30cm |
| price | 价格 | 选填 | 99.99 |
| stock_quantity | 库存数量 | 选填 | 100 |
| category | 产品分类 | 选填 | 电子产品 |
| brand | 品牌 | 选填 | 示例品牌 |
| supplier | 供应商 | 选填 | 示例供应商 |
| description | 产品描述 | 选填 | 产品描述 |

### 3.3 上传文件

1. 将填写好的 Excel 文件拖拽到上传区域，或点击上传区域选择文件
2. 系统会自动校验文件格式（仅支持 .xlsx 和 .xls）
3. 文件大小不能超过 10MB
4. 确认文件后，点击 **"开始导入"** 按钮

### 3.4 查看导入结果

导入完成后，系统会显示详细的导入报告：

- **总计** - 文件中的数据总行数
- **成功** - 成功导入的商品数量
- **失败** - 导入失败的商品数量

如有失败记录，系统会列出具体的错误原因：
- 缺少必填字段
- 数据格式不正确（如文本填入数字字段）
- SKU 重复
- 数值必须大于 0

---

## 四、手动录入操作步骤

### 4.1 进入手动录入页面

1. 进入 **"产品上传"** 页面，选择 **"手动录入"** 标签

### 4.2 填写表单

按照表单提示填写商品信息：

- 带 **红色星号(*)** 的字段为必填项
- 尺寸和重量单位分别为 **厘米(cm)** 和 **千克(kg)**
- 价格和库存数量可选填

### 4.3 字段验证

系统提供实时字段验证：
- SKU 不能为空，长度不超过 50 个字符
- 产品名称不能为空
- 长度、宽度、高度、毛重必须大于 0
- 价格不能为负数
- 库存数量不能为负数

如果填写有误，字段下方会显示红色错误提示。

### 4.4 提交保存

1. 确认所有必填字段已正确填写
2. 点击 **"保存产品"** 按钮
3. 提交成功后，系统会显示成功提示："产品添加成功！系统已自动完成单位转换。"
4. 点击 **"重置表单"** 可清空表单重新填写

---

## 五、单位转换说明

### 5.1 自动转换

所有商品信息在入库时，系统会自动进行单位转换：

| 原始单位 | 转换后单位 | 转换公式 |
|----------|------------|----------|
| 厘米 (cm) | 英寸 (inch) | 1cm = 0.393701inch |
| 千克 (kg) | 磅 (lb) | 1kg = 2.20462lb |

转换结果保留 **2位小数**。

### 5.2 转换示例

| 原始值 | 转换结果 |
|--------|----------|
| 10 cm | 3.94 inch |
| 25.4 cm | 10.0 inch |
| 1 kg | 2.2 lb |
| 1.5 kg | 3.31 lb |

## 六、双版本查看操作

### 6.1 搜索商品

1. 进入 **"产品搜索"** 页面
2. 在搜索框中输入 SKU 关键词（留空可查看全部）
3. 点击 **"搜索"** 按钮

### 6.2 切换单位版本

在搜索结果表格上方，可以看到单位版本切换按钮：

- **原始单位 (cm / kg)** - 显示厘米和千克
- **转换单位 (inch / lb)** - 显示英寸和磅

点击即可在两种单位之间切换。

### 6.3 查看商品详情

1. 在搜索结果表格中，点击商品行右侧的 **"详情"** 按钮
2. 页面下方会展开详情面板，同时展示：
   - **原始单位** - 厘米/千克数据
   - **转换单位** - 英寸/磅数据
   - **其他信息** - 型号、规格、价格、库存等
3. 如果已转换，面板底部会显示具体的转换计算公式和转换时间
4. 再次点击 **"收起"** 可关闭详情面板

### 6.4 转换状态标识

每行商品右侧会显示转换状态：
- **已转换**（绿色标签）- 已完成单位转换
- **未转换**（黄色标签）- 尚未进行单位转换

## 七、常见问题与解决方案

### Q1: Excel 导入提示"缺少必填列"

**原因**：Excel 表头名称与模板不一致。

**解决方案**：
- 请使用系统提供的标准模板
- 确保表头名称为：sku, name, length_cm, width_cm, height_cm, gross_weight_kg
- 也可使用中文表头：SKU, 产品名称, 长度(cm), 宽度(cm), 高度(cm), 毛重(kg)

### Q2: 导入提示"SKU 已存在"

**原因**：该 SKU 在系统中已经存在。

**解决方案**：
- 检查 Excel 文件中是否有重复 SKU
- 如需更新已有商品，请联系管理员

### Q3: 数值字段导入失败

**原因**：Excel 单元格格式为文本，或包含非数字字符。

**解决方案**：
- 确保数值字段为数字格式
- 删除前后空格和特殊字符
- 检查是否有隐藏字符

### Q4: 单位转换结果不准确

**原因**：系统使用标准转换公式，结果保留2位小数。

**解决方案**：
- 转换公式：1cm = 0.393701inch，1kg = 2.20462lb
- 由于四舍五入到2位小数，可能存在微小误差，属正常现象
- 如需更高精度，请联系开发团队

### Q5: 文件上传失败

**原因**：文件格式或大小不符合要求。

**解决方案**：
- 仅支持 .xlsx 和 .xls 格式
- 文件大小不能超过 10MB
- 如文件过大，请拆分为多个文件导入

## 八、注意事项

1. **数据备份**：建议在批量导入前备份现有数据
2. **SKU 唯一性**：SKU 是商品的唯一标识，不可重复
3. **单位统一**：导入时尺寸统一使用厘米(cm)，重量统一使用千克(kg)
4. **权限要求**：上传商品需要 `product.create` 权限，如无法操作请联系管理员
5. **网络稳定**：批量导入时请保持网络连接稳定
